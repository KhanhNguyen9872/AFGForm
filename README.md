# How to build
1. Download and install `Visual Studio 2022`
2. Checked `.NET desktop development` 
3. Download this repo [here](https://github.com/KhanhNguyen9872/AFGForm/archive/refs/heads/main.zip)
4. Open `AFGForm.sln` file
5. Build and run
